# ESP32 WiFi Scanner
ESP32 WiFi Scanner — это проект, который позволяет ESP32 сканировать все доступные WiFi точки доступа и выводить информацию о них в последовательный монитор. Этот код использует библиотеку `WiFi` для работы с WiFi на ESP32.

## Описание
Проект включает в себя скетч для Arduino IDE, который позволяет ESP32 сканировать все доступные WiFi точки доступа и выводить информацию о них в последовательный монитор. Информация включает SSID, силу сигнала (RSSI), BSSID и тип шифрования каждой сети.
## Требования
- Платформа Arduino IDE версии 1.x
- ESP32 модуль
- Компьютер с поддержкой USB
## Установка
1. Установите [Arduino IDE](https://www.arduino.cc/en/software) (если еще не установлен).
2. Скачайте и установите библиотеку `WiFi` для ESP32, если она не включена по умолчанию в вашей версии Arduino IDE.
3. Подключите ESP32 к компьютеру через USB.
4. Откройте скетч в Arduino IDE и загрузите его на ESP32.
## Использование
1. После загрузки кода на ESP32, он будет сканировать все доступные WiFi сети каждые 10 секунд и выводить информацию о каждой сети в последовательный монитор.
2. Вы можете изменить интервал сканирования, отредактировав строку `delay(10000)` в функции `loop()` на желаемый интервал времени в миллисекундах.
